# Changes on branch `gradle-scan-reports`
Document any changes on this branch here.
### Added
- Gradle targets for image scanning:
  - `gradle grype:<project>` - grype scan of project
  - `gradle grypeReport` - summary of all projects
  - `gradle grypeFullReport` - summary of all images running in current local cluster
  - `gradle trivy:<project>` - trivy scan of project
  - `gradle trivyReport` - summary of all projects
  - `gradle trivyFullReport` - summary of all images running in current local cluster
