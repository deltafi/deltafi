## [2.36.0] - 2025-11-17

### Added
- Added a `deltafi.storage.url` setting to the `values.yaml` to allow pointing to an external content storage system
- Added a `deltafi.storage.snowball.enabled` setting to the `values.yaml` to allow using snowball uploads to external content storage systems when they support it. By default, this is set to the value of `enabled.minio`
- TUI: Python plugin code generation
  - `deltafi plugin generate --python <plugin name>`
  - `deltafi plugin generate action <plugin name> <action name>`

### Changed
- Changed the helm chart to skip the `deltafi-fast-delete` deployment when running with external content storage
- Changed the `Kubernetes Resource Check` configuration to account for disabled services
- Changed the status checks to skip checking content storage when running with external content storage
- Changed content related system properties to be disabled when running with external content storage 
- TUI: New Java plugin code generation
  - `deltafi plugin generate --java <plugin name>`
  - `deltafi plugin generate action <plugin name> <action name>`
- GraphQL endpoint `getPropertySets` now has an optional `splitByGroup` parameter (defaults to false)
- UI System Properties page now shows core properties in groups

### Fixed
- nodemonitor container image using latest clean docker client (CVE mitigation)
- Updated the TUI Makefile image for golang to `golang:1.25.4`

### Tech-Debt/Refactor
- Updated the toString in `DeltaFileFlow` and `DeltaFileFlowInput` classes to avoid circular references

### Upgrade and Migration
- Improved CVE mitigation by using latest chainguard kubectl in clustermonitor
- New docker images:
  - deltafi/minio:RELEASE.2025-10-15T17-29-55Z-2
  - valkey/valkey:8.1.4
  - deltafi/grafana:12.2.1-0
  - grafana/promtail:3.5.8
  - grafana/loki:2.9.17
  - nginx:1.29.3-alpine3.22
  - deltafi/docker-web-gui:1.0.2-4
  - deltafi/timescaledb:2.19.3-pg16-2
- When using external content storage the operator is responsible for the bucket creation and life cycle management. DeltaFi does not attempt to create the bucket or set the age-off when using external content storage.
- Upgrade to Java 21.0.9
- Upgrade to Go 1.25.4
- Valkey upgrade to 9.0.0
